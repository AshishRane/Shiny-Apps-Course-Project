Brief introduction to ggplot2 for shinyapp

App Overview

This app is to provide brief introduction to ggplot2 in R. It has 5 parameters to select variable x, variable y, color variable, facet wrap variable and type of the plot. The dataset used is the mpg dataset in R for 38 models in car.

Usage

  1. Select any of the 5 parameters
  2. The plot & ggplot command will change automatically

Code Access

The ui.R, server.R and other related files are available at this public git repository https://github.com/welcheb/DevelopingDataProductsProject
